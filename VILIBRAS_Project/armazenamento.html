ARMAZENAMENTO

Armazenamento

Descrição: Local para guardar dados de forma organizada, permitindo o acesso posterior.

Arquivo

Descriçãp: coleção de dados armazenados sob um nome único.

Backup

Descrição: Backup é uma cópia de segurança de dados essencial para proteção.

Banco de Dados

Descrição: sistema organizado para armazenar, gerenciar e recuperar informações.

Bluetooth

Descrição: comunicação sem fio entre dispositivos.

Compactar

Descrição: processo de redução do tamanho de um arquivo ou pasta.

Criptografia

Descrição: proteção de informações por de ténicas de codificação.

Dados

Descrição: informações ou fatos brutos que, por si só, não têm significado. Quando processados e interpretados, os dados adquirem significado e se tornam informações.

Decodificação

Processo de reverter a codificação ou criptografia aplicada aos dados.

Descompactar

Descompactar é restaurar o arquivo ou pasta à sua forma original após ter sido compactado

Disquete

Disquete é um meio de armazenamento magnético removível, amplamente utilizado nas décadas de 1980 e 1990,

Documento

Registro ou representação de informações, frequentemente em formato escrito ou eletrônico, organizado de maneira a comunicar ou armazenar dados.

Download

Descrição: obter dados ou arquivos da internet para armazenamento no dispositivo local.

Extensão de Arquivo

Sequência de caracteres no final de um nome de arquivo que indica o tipo de arquivo e geralmente está associada ao formato do conteúdo dentro do arquivo.

PDF

PDF (Portable Document Format) é um formato de arquivo projetado para apresentar documentos de maneira independente de dispositivo ou software.

Repositório

Local ou sistema onde algo é armazenado.

Servidor

Computador dedicado ao armazenamento e gerenciamento de dados em rede.

Upload

Descrição: enviar dados ou arquivos do dispositivo local para um servidor ou plataforma online.
